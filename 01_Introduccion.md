# Introducción

El Diseño y Programación Web es una disciplina que combina creatividad, diseño y tecnología para la creación de sitios web y aplicaciones digitales. Esta área permite desarrollar soluciones que facilitan la comunicación, el acceso a la información y la interacción de los usuarios mediante plataformas digitales.

En la actualidad, Internet se ha convertido en una herramienta fundamental para empresas, instituciones y personas, generando la necesidad de contar con profesionales capaces de diseñar y desarrollar páginas web modernas, funcionales y adaptadas a diferentes dispositivos.

La carrera de Diseño y Programación Web integra conocimientos de diseño gráfico, experiencia de usuario (UI/UX), programación y herramientas digitales, permitiendo crear proyectos que unen la parte visual con la parte tecnológica.

## Objetivo

El objetivo de este proyecto es presentar los principales aspectos relacionados con la carrera de Diseño y Programación Web, incluyendo su definición, áreas de trabajo, herramientas fundamentales, perfil del estudiante y profesional, así como las oportunidades laborales que ofrece esta disciplina.

## Importancia del Proyecto

Este proyecto permite conocer la importancia del Diseño y Programación Web dentro del mundo digital actual, además de aplicar el trabajo colaborativo mediante el uso de Git y GitHub como herramientas para organizar y gestionar proyectos en equipo.
