# Conclusiones

El Diseño y Programación Web es una disciplina que combina creatividad, tecnología y conocimientos de desarrollo para crear soluciones digitales funcionales y atractivas. A través de esta carrera se integran áreas como el diseño gráfico, la experiencia de usuario y la programación, permitiendo desarrollar sitios web y aplicaciones adaptadas a las necesidades actuales.

Durante el desarrollo de este proyecto se identificaron las principales áreas de la carrera, las herramientas fundamentales utilizadas por los profesionales y las habilidades necesarias para desenvolverse en el campo laboral. Estos elementos demuestran la importancia de contar con profesionales capaces de unir el aspecto visual con la parte tecnológica.

Asimismo, se pudo reconocer que el Diseño y Programación Web cuenta con diversas oportunidades laborales debido al crecimiento constante de la tecnología y la necesidad de las organizaciones de tener presencia en Internet.

Finalmente, este proyecto permitió aplicar el trabajo colaborativo mediante el uso de Git y GitHub, herramientas que facilitan la organización, control de cambios y desarrollo de proyectos en equipo, fortaleciendo las habilidades necesarias para el ámbito profesional.
